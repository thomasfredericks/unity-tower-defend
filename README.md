# unity-tower-defend


Exemple présenté dans le cours Objets interactifs.

## Structure

- `arduino-tower-defend` : code Arduino PlatformIO
- `unity-tower-defend` : code Unity

## Unity

### Version Unity

- 2022.3.62f2

### Scène principale 

- Assets > tower-defend > Scenes

### Instructions

- Utiliser les touches ↑ et ↓ du clavier pour déplacer la palette de gauche.
